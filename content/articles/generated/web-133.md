1. 工程自动化约束为主，局部安全降级为辅

2. CSS 维度的现代兼容痛点

   1. 通过 **PostCSS** 拦截所有样式资产。

      使用 `autoprefixer` 自动依据目标矩阵追补厂商前缀（`-webkit-`, `-moz-`）。

      使用 `postcss-preset-env` 允许开发编写前沿的 CSS 特性，并在编译期自动将其编译为旧浏览器认识的阶梯语法。

3. JavaScript 维度的现代兼容痛点

   1. 一些新的语法不支持，如可选链等新规范
   2. 一些API缺失
   3. 使用babel进行处理
   4. **自建阿里开源的 Polyfill.io 服务（网络动态下发）**：页面 HTML 的头部挂载一个中立域名的 `<script src="https://polyfill.io/v3/polyfill.min.js"></script>`。该服务器会分析请求头（User-Agent）。如果是最新的 Chrome 浏览器，服务器返回一个**空文件（0 字节）**；如果是古老的 iOS 11，服务器会在后台动态打包出其缺失的所有垫片吐给它。利用服务器算力，实现对用户端体积的完美瘦身。
