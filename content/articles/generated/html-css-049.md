| **隐藏手段**             | **脱离标准流？** | **空间占位保留？** | **会否引发回流（Reflow）？** | **会否引发重绘（Repaint）？** | **鼠标/点击事件能触发？** | **支持 CSS Transition 渐变动效？** |
| ------------------------ | ---------------- | ------------------ | ---------------------------- | ----------------------------- | ------------------------- | ---------------------------------- |
| **`display: none`**      | **是**           | 否                 | **是（全栈级重构）**         | 是                            | 否                        | 否（瞬间突变死锁）                 |
| **`visibility: hidden`** | 否               | **是**             | 否                           | **是（仅局部图层）**          | 否                        | **是（支持平滑离散过渡）**         |
| **`opacity: 0`**         | 否               | **是**             | 否                           | 否（开启GPU则 0 渲染开销）    | **是（事件照常触发）**    | **是（最完美支持）**               |
| **移出视口**             | **是**           | 否                 | 仅在首次渲染移出时触发一次   | 是                            | 否                        | 否                                 |
| **`clip-path` 裁切**     | 否               | **是**             | 否                           | 是                            | 否                        | 是                                 |
