| **特性**             | **AMD (RequireJS)**                                          | **CMD (SeaJS)**                                              |
| -------------------- | ------------------------------------------------------------ | ------------------------------------------------------------ |
| **依赖声明**         | **前置声明**：在定义模块的开头必须列出所有依赖数组。         | **就近声明**：在用到的地方直接局部 `require` 即可。          |
| **执行时机**         | **提前执行**：依赖模块异步下载完后**立刻执行**（不管后续用不用）。 | **延迟执行**：依赖模块下载后不执行，直到运行到 `require` 这一行才**首次执行**。 |
| **对用户体验的影响** | 首屏加载可能稍慢，但后续模块内部逻辑调用时响应极快。         | 模块初始化快，但运行到某个深度依赖时，可能因为临时执行而产生微小的滞后。 |
| **API 辨识度**       | `define(['dep1'], function(d1) {})`                          | `define(function(require, exports, module) {})`              |
