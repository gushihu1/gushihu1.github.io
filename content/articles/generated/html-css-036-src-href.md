| **维度**     | **href (以 <link> 为例)**                      | **src (以 <script> 为例)**                        |
| ------------ | ---------------------------------------------- | ------------------------------------------------- |
| **英文全称** | Hypertext Reference（超文本引用）              | Source（资源来源）                                |
| **本质作用** | **引入**：建立通道，使当前文档与目标资源关联。 | **替换**：将外部资源下载并嵌入/替换当前标签。     |
| **DOM 解析** | **不阻塞** DOM 树的解析，实行异步并行下载。    | **强阻塞** DOM 树的解析，必须下载并执行完才继续。 |
| **作用结果** | 改变当前文档的**样式或建立跳转**（外附）。     | 成为当前文档的**核心行为或物料**（内聚）。        |
