| **进程类型**                    | **数量**                         | **核心职责**                                                 | **运行环境**      |
| ------------------------------- | -------------------------------- | ------------------------------------------------------------ | ----------------- |
| **主进程 (Main Process)**       | **有且仅有一个**                 | 管理应用生命周期、创建/控制浏览器窗口、调用操作系统原生 API 。 | **Node.js 环境**  |
| **渲染进程 (Renderer Process)** | **多个**（每个窗口对应一个进程） | 负责 UI 界面的渲染和交互 。为了安全，默认处于沙箱隔离状态 。 | **Chromium 环境** |

1. 核心底座：双核驱动。Electron 的强悍之处在于它同时集成了两个成熟的引擎 ：

   - **Chromium**：负责页面的渲染、DOM 解析和 UI 交互，为前端开发者提供标准的 Web 运行环境 。

   - **Node.js**：负责打破浏览器的沙箱限制，赋予应用访问操作系统底层（如文件系统、网络请求、硬件交互等）的能力 。
