| **特性**     | **var**                    | **let**                | **const**                                |
| ------------ | -------------------------- | ---------------------- | ---------------------------------------- |
| **作用域**   | 函数作用域                 | 块级作用域             | 块级作用域                               |
| **变量提升** | 支持（初始化为 undefined） | 支持（进入暂时性死区） | 支持（进入暂时性死区）                   |
| **重复声明** | 允许                       | 不允许                 | 不允许                                   |
| **修改值**   | 可修改                     | 可修改                 | 不可修改（基本类型）/ 引用不可变（对象） |
| **全局属性** | 会挂载到 window            | 不会挂载到 window      | 不会挂载到 window                        |
