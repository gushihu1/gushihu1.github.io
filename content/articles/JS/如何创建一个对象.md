1. 对象字面量 (Object Literal)

   - **优点**：写法简单，执行效率高。

   - **缺点**：无法批量生产，无法实现逻辑复用。

   - ```javascript
     const user = {
       name: '张三',
       age: 25,
       sayHi() { console.log('你好'); }
     };
     ```

     

2. 构造函数 (Constructor)

   - ```javascript
     function User(name, age) {
       this.name = name;
       this.age = age;
     }
     User.prototype.sayHi = function() { console.log(this.name); };
     const user1 = new User('张三', 25);
     ```

     

3. 工厂模式 (Factory Pattern)

   - ```javascript
     function createUser(name, age) {
       return {
         name,
         age,
         sayHi() { console.log(this.name); }
       };
     }
     const user1 = createUser('张三', 25);
     ```

4. ES6 Class (推荐方案)

   - ```
     class User {
       constructor(name, age) {
         this.name = name;
         this.age = age;
       }
       sayHi() {
         console.log(`我是${this.name}`);
       }
     }
     ```
