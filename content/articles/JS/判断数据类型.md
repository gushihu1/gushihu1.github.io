1. `Object.prototype.toString.call()`：终极方案（最推荐）
   - **原理**：调用 Object 原型上的 `toString` 方法，它会返回一个固定格式的字符串 `[object Type]` 。
   - **优点**：**最精准、最通用**，能准确区分 `Array`, `Null`, `Date`, `RegExp` 等所有内置类型 。
2. `typeof`：最快但有缺陷
   - **适用场景**：判断**基本数据类型**（除 `null` 外）和 `function` 。
   - **缺陷**：
     - `typeof null === 'object'`：这是 JS 历史遗留的二进制判断 Bug 。
     - **无法区分引用类型**：数组、对象、正则、日期等统统返回 `'object'` 。
3. 其他的
   1.  `instanceof`：基于原型链
   2. `constructor`：查看构造函数
4. 特殊
   1. **判断数组**：优先使用 ES6 的 `Array.isArray()`，它专门解决了 Iframe 环境下的判断问题 。
