| **特性**                   | **普通函数 (function)**           | **箭头函数 (=>)**                            |
| -------------------------- | --------------------------------- | -------------------------------------------- |
| **`this` 指向**            | **动态绑定**：指向调用它的对象 。 | **静态绑定**：捕获定义时所处环境的 `this` 。 |
| **构造函数能力**           | 可以作为构造函数（使用 `new`） 。 | **不可以**：没有 `[[Construct]]` 方法 。     |
| **`arguments` 对象**       | 拥有自己的 `arguments` 对象。     | **没有**：需使用剩余参数 `...args` 替代。    |
| **原型属性 (`prototype`)** | 拥有 `prototype` 属性 。          | **没有** `prototype` 属性。                  |
| **`yield` 关键字**         | 可以作为生成器函数（Generator）。 | 不能使用 `yield` 关键字。                    |
