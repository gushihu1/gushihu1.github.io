##### 内容

1. **`content`（内容区）**：呈现真实文本、图片等内容的区域。
2. **`padding`（内边距）**：内容区与边框之间的透明区域，会受到背景色（background-color）的浸染。
3. **`border`（边框）**：包裹在 padding 外层的线条。
4. **`margin`（外边距）**：盒子与周围其它盒子之间的空白间距，它是完全透明的，不吸收背景色。

##### 类型

1.  标准盒子模型 (`box-sizing: content-box`) —— 浏览器默认
   1. width = content
2. IE 怪异盒子模型 (`box-sizing: border-box`) —— 工业界标准
   1. width = content + padding + border
3.  一般通过初始化样式文件对box-sizing进行统一，让不同浏览器展示相同效果
