1. `calc()` 允许在声明 CSS 属性值时执行数学运算
   1. **支持运算符**：`+`（加）、`-`（减）、`*`（乘）、`/`（除）。
   2. **单位混合**：这正是 `calc()` 的杀手锏。你可以在同一个计算表达式中混合使用完全不同的单位，例如 `px`、`rem`、`vw`、`%`、`em` 等。
2. `+` 和 `-` 运算符的两侧**必须保留空格**。
